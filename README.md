## 🚀 Featured Projects

### [Career-Pilot](https://github.com/vishnup102002/Career-Pilot) — Autonomous AI Job-Hunting Agent
> Python · LangGraph · FastAPI · MCP · Docker · Groq

A 4-agent pipeline (Research → Scout → Writer → Alert) that parses your resume, scrapes matching jobs via a custom FastMCP + Playwright server, scores each listing out of 5, and delivers a daily digest via SendGrid — fully automated, runs unattended.

**Key decisions:** Single batched Groq API call for all job matching. SQLite deduplication across runs. Conditional exits to avoid wasted compute. Deployed on HuggingFace Spaces via Docker.

🔗 [Live Demo](https://huggingface.co/spaces/Vishnuporkulath/career-pilot)
